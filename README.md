# Datanomics Group 8 Project

## Project Structure

- `src/` — Main source code for all project tasks
    - `snack_drink_shop/` — Task 1: Snack & Drink Shop program
    - `grocery_cart/` — Task 2: Grocery Cart program
    - `todo_manager/` — Task 3: To-Do List Manager
    - `movie_booking/` — Task 4: Movie Ticket Booking Simulation
    - `quiz_game/` — Task 5: Quiz Game
- `tests/` — Test scripts for each module
- `docs/` — Documentation, guides, and planning

## Collaboration Guidelines
- Each member should work in their assigned folder/module.
- Use `docs/` for shared notes, requirements, and planning.
- Place all test scripts in `tests/`.
- Use clear commit messages and branch names.

Welcome, Group 8! Start by picking a task and working in the corresponding folder. Happy coding!
